① ROUND 1 · 20 MIN · TAs WALK

Round 1 — terminal + editor.

You will

  • Install VS Code (or instructor-specified editor).
  • Open a terminal — built-in panel inside the editor counts.
  • Navigate one folder up, one folder down. Edit a file. Save.
  • Run the verification command at your own machine, aloud.

Verify

$ echo "hi" $ pwd // outputs the line + your home path

05 / 12
② ROUND 2 · 15 MIN · COMMIT IS THE GOAL

Round 2 — runtime + git.

You will

  • Install Python (or course language). Verify the version.
  • Run a one-line `print("hi")` script. The room sees output.
  • Initialize a git repo. Commit a one-line README.
  • Push to a new GitHub repo. The repo can be empty — the act of pushing is the lesson.

Verify

$ python --version $ git --version $ git log --oneline // shows at least 1 commit

④ FIRST NOTE · 15 MIN · USE WHAT YOU INSTALLED

First note. First source.

Installing tools is not a capability. Using them is. We're doing both today, in the same room, with help at your elbow.

First note · Weeks/W03-setup-quest.md

  • One sentence on what you installed.
  • Save. That's it.

First source · Sources/[topic].md

  • Find one credible source on something you're curious about.
  • From an IU database. NOT Google.
  • Capture title, link, and one sentence on why you trust it.
