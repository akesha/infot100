Two are wrong. One is right. All three sound equally confident. Pick the one you'd trust — don't tell anyone yet.
> What does this Python code print? — print([1,2,3] * 2)
GPT-Style
It prints [2, 4, 6]. The * 2 multiplies each element by 2 — element-wise multiplication, the standard list behavior.
Claude-Style
[1,2,3] * 2 evaluates to [1, 2, 3, 1, 2, 3] — multiplying a list by an int repeats it. (For element-wise, you'd need numpy.)
Mixed-bag
It prints [1, 2, 3] twice on separate lines because * 2 is shorthand for two prints.

⑤ PAIR AUDIT · 20 MIN

Trade prompts.
Rate each other.

  1. Find a partner
    Different major if possible. Surfaces real framing differences.
  2. Read theirs slowly
    Out loud is fine. Score on four axes: SPECIFIC · SCOPED · VERIFIABLE · ETHICAL.
  3. Discuss the lowest score
    Not the highest. The lowest one is where the prompt is weakest — and where the conversation is most useful.
